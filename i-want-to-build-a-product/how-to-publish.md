# How to make this public

This folder is now a small web app. The simplest way to publish it is Vercel.

## Easiest path with Vercel

1. Go to https://vercel.com
2. Sign in.
3. Create a new project.
4. Import or upload this folder.
5. Keep the default settings.
6. Click Deploy.
7. Vercel will publish it and give you a public link.

You do not need an App Store account for this version. People can open the link on their phone. On iPhone, they can tap Share, then Add to Home Screen.

## What to publish

Publish the whole folder:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `service-worker.js`
- `assets/icon.svg`
- `vercel.json`

The documents are useful for you, but the app can run without them:

- `product-blueprint.md`
- `database-schema.sql`
- `README.md`

## Before sharing widely

Replace the demo venue entries with real verified Tokyo places. The current database is good for testing the product idea, not for sending people out into the city yet.
