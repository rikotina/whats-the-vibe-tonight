# What's the Vibe Tonight?

A first working MVP for a vibe-first Tokyo discovery app.

## What is built

- Mobile-first discovery app
- Curated starter Tokyo place database
- Vibe quiz with mood, neighborhood, budget, outfit, social energy, time, and reservation tolerance
- Weighted recommendation engine returning three different result roles
- Saved places using local browser storage
- Local admin surface for adding and editing places
- Shareable query-string state for result pages

## Open it locally

Open `index.html` in a browser.

For the installable web-app behavior, run it from a local web server or publish it online. The service worker does not run from a plain file link.

## Publish it

See `how-to-publish.md`.

## Notes

The included place database is a demo seed for product development. Before public launch, replace or verify each venue record with current address, hours, reservation links, photos, and editorial review.
